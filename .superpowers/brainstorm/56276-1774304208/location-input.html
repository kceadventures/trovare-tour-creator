B: Segmented Control

iOS-style pill toggle. More visual weight, clearer active state. Selected address shows in a green-tinted confirmation box with both name and coordinates. Feels more "appy".

Behavior (both options)

  • Search tab: Debounced Nominatim autocomplete (300ms). Select result → auto-fills lat/lng. Shows resolved coords as confirmation.
  • Coordinates tab: Same raw lat/lng number inputs as today. Unchanged.
  • Default tab: "Search" when coords are empty, "Coordinates" when coords already populated (e.g. from GPX).
  • Switching tabs: Preserves values — switching to Coordinates after a search shows the auto-filled values, editable.
  • Same component reused in both stop cards and new region creation dialog.
