Component Animations

🔘 Button
  • Hover: scale(1.02), spring ease
  • Press: scale(0.97), instant feel
  • Focus ring: animated opacity
  • Loading state: smooth width transition
🃏 Card
  • Mount: fade in + slide up (staggered)
  • Hover: subtle lift (y: -2, shadow increase)
  • Choice cards: spring scale on hover
💬 Dialog
  • Overlay: fade in (opacity 0→1)
  • Content: scale(0.95)→1 + fade, spring
  • Exit: reverse with faster timing
  • Backdrop blur animates in
📋 Select / Dropdown
  • Content: scale(0.96)→1 + fade from origin
  • Items: staggered fade-in (20ms delay)
  • Exit: fast fade + scale down
  • Scroll buttons: opacity transition
📊 Progress
  • Bar: spring-animated width changes
  • Value label: number counting animation
  • Track: subtle pulse while active
🏷️ Badge
  • Mount: pop in (scale 0→1, spring)
  • Exit: shrink + fade out
  • Count change: layout animation
📑 Tabs
  • Active indicator: layoutId sliding pill
  • Content: crossfade between panels
  • Direction-aware slide (left/right)
📥 Input / Textarea
  • Focus: border color spring transition
  • Error shake: horizontal spring wiggle
  • Textarea: smooth height animation
